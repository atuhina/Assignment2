# Assignment 2 

## Overview
For this assignment I implemented 4 features, such as:

Dot Product:
Used to check if the totem is facing the correct direction.

Lerp:
Used to smoothly open the door.

Particles:
Particles appear when the puzzle is solved.

Sound:
A sound plays when the puzzle is completed.

## How to Play
- Move the player using the keyboard
- Go by the totem 
- Rotate the totem by pressing E
- When it is facing the correct direction:
  - The color changes
  - Particles appear
  - The door opens
  - A sound plays
